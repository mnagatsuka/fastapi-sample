# Dev

Activate
```bash
source .venv/bin/activate
```

Deactivate
```bash
deactivate
```

Run dev
```bash
uv run fastapi dev
```

Lint
```bash
ruff check
```

Format
```bash
ruff format
```


Local Server
`http://127.0.0.1:8000`
